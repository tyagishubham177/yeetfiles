# YeetFiles Icon Brand Spec

## Selected direction

The icon now uses an abstract motion-shard mark instead of the old layered document metaphor.

The selected mark is designed to communicate:

- forward motion
- controlled cleanup
- quiet confidence
- premium utility rather than file-manager literalism

## Symbol logic

The mark is built from three shards:

- a rear shard that hints at queue progression
- a dominant lower diagonal shard that carries the visual momentum
- a smaller front shard that completes the forward-cut silhouette

Together, the negative space suggests a subtle `Y` structure without turning the icon into a lettermark first.

## Color system

- tile background start: `#0B1220`
- tile background end: `#17263B`
- primary mark: `#F4F7FB`
- restrained highlight shard: `#B8C7DA`

## Usage notes

- Keep the mark large and centered.
- Do not add document corners, folders, or camera metaphors around it.
- Prefer the dark tile for launcher and splash contexts.
- Prefer the transparent foreground mark for Android adaptive icon use.
- Use the monochrome export when checking tiny-size clarity or printing.
